# DPDP Compliance Checker

A Python Tkinter-based DPDP Compliance Checker with Excel & PDF export.

## Features
- Collects user details (Institution, Name, Email)
- Numbered Yes/Partial/No questionnaire
- Calculates total score, compliance %, and risk level
- Saves results automatically to timestamped Excel & PDF
